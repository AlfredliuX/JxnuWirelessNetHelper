### 介绍：<br>
这是一款专为师大人开发的app,主要为了方便连接校园网后的一系列操作.<br>
主要功能：<br>
1）连接wifi后自动登录<br>
2）保持自服务的登录状态<br>
3）快速注销<br>

### 常见问题：<br>
1.如果你觉得经常莫名其妙的自动登录，可以关闭自动登录的功能<br>
2.请不要禁止软件的自启动，否则无法自动登录<br>
3.连接wifi时，部分机型需要同时关闭数据连接，否则无法自动登录<br>
